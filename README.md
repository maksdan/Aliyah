# Aliyah

A React Native (Expo) app for daily Torah aliyah reading. Shows today's parasha portion in Hebrew, English, or both, fetched from [Sefaria](https://www.sefaria.org/).

## Features

- Daily aliyah based on the current day of the week
- Hebrew and English text from Sefaria
- Mark today's reading as complete
- Weekly notification reminders

## Getting Started

```bash
npm install
npm start
```

Press `i` to open in the iOS simulator, or scan the QR code with Expo Go.

## Tech Stack

- Expo ~54
- React Native
- TypeScript
- Sefaria API
